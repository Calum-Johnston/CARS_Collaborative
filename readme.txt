The following is a readme.txt file for recommender-system.py.

In order to run the file, simply double click the python file to execute.
To sign in, simply select a user ID from the range 1001 - 1042, any ID within the range will be accepted by the system.

Please note:
- Python 3.7.4 was used to write the program
- Additional libraries were utilized: Numpy, Scipy, Sklearn and Pandas
- The dataset is all contained in the folder: Dataset, which contains 3 csv files: ContextualRatings_InCarMusic.csv, 
  MusicData_InCarMusic.csv, UserData_InCarMusic.csv
- The evaluating function takes some time to run, please be patient!
